Links the database to the player entry screen.

#### Tkinter required

Debian:  
sudo apt-get install python3-tk

MacOS:  
brew install python-tk

#### psycopg2 required

Debian:  
sudo apt install python3-psycopg2

MacOS:  
brew install postgresql  
pip install psycopg2
